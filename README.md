# 🔐 Java RC4 Cipher Project

This project is a Java-based implementation of the RC4 Cipher Algorithm used for encryption and decryption of text messages. It converts plain text into cipher text using a secret key and decrypts it back to the original message using the same key.

---

## 📌 Objective

To design and implement a secure RC4 Cipher system in Java that demonstrates symmetric key cryptography for educational purposes.

---

## 🚀 Features

- Text Encryption using RC4 Algorithm
- Text Decryption using Same Secret Key
- Java Console Demonstration
- Servlet-based Web Integration
- Frontend and Backend Support

---

## ⚙️ How It Works

1. User enters plain text.
2. Secret key is given.
3. RC4 algorithm generates encrypted cipher text.
4. Same key is used to decrypt the cipher text.
